1: whoami
   groups
 -This command displays my currently logged-in username and the groups my account belongs to, confirming my user identity on the system.
2: pwd
   ls -l
  -The pwd command shows my current working directory, and ls -l lists all files and directories in long format including permissions and ownership.
3: echo "Linux user environment verified" > user_info.txt
  -This command creates a file named user_info.txt and writes a confirmation message into it.
4: wc -m user_info.txt
  -This command counts the total number of characters in the file to verify its contents.
5:  mkdir --help
  -This command creates a directory or directories if they do not exist. An option -p can be added to also create parent directory.
6: ls ~
  -This command lists all files and directories in my home directory in alphabetical order.
7: grep "admin" log.txt
  -This command searches for the word “admin” inside log.txt and displays only matching lines.
8: uname -r
  -This command displays the currently running Linux kernel version.
9: ping www.google.com
  -This command sends ICMP packets to Google to verify that the system has network connectivity.
10: uptime
  -The output shows how long the system has been running, the number of logged-in users, and the system load averages.
